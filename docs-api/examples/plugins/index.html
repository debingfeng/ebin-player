插件开发指南

插件结构

  • 继承 BasePlugin 基类
  • 实现 meta 元数据
  • 定义配置接口
  • 实现生命周期钩子

生命周期

  • onInit - 插件初始化
  • onStart - 插件启动
  • onConfigChange - 配置变更
  • onDestroy - 插件销毁

API接口

  • ctx.player - 播放器实例
  • ctx.logger - 日志记录器
  • ctx.on/off - 事件监听
  • ctx.storage - 数据存储
